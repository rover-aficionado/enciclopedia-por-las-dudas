¿CÓMO LO HE HECHO?

en verdad solo he tenido que aprender unos comandos rápidos para iniciar a escribir y para terminar que es importante pero tódo lo hace el ordenador. Incluso se puede hacer escuchando música, ahora estoy escuchando "Ocean Drive" de Duke Drive, muy recomendada.

El código está hecho a partir de Visual Studio Code que me lo descargué en su día para intentar aprender Python (en teoría era fácil) pero entre unas cosas y otras lo dejé abandonado. A cambio aprendí a usar el cmd de Windows para acceder, abrir, borrar archivos; abrir aplicaciones y calcular operaciones simples desde allí. Para visualizar el resulado (que sé que no es perfecto por que no he estudiado css para editarlo) he usado una extensión de Visual Studio Code llamada Live Server que me va actualizando la página según la voy guardando en un entorno simulado de Microsoft Edge.

Para aprender esto no he tenido más remedio que usar un tutorial gratuito de YouTube y mucha paciencia.
